Tema 1: Flexbox

Tiene como objetivo proporcionar una manera mas eficiente de diseñar, alinear y distribuir espacio entre los elementos.

La idea principal del flexbox es otorgar al contenedor la capacidad de modificar el ancho alto y el orden de sus elementos para llenar mejor el espacio disponible (principalmente para adptarse al tipo de pantalla)

Lo mas importante es que el diseño de flexbox es independiente de la direccion en el lugar de los diseños regulares (bloque que se basa verticalmente y en la linea que se basa horizontalmente.)

Propiedades

  • display: Define un contenedor como un contenedor flex o un elemento flex en el flujo de línea.

  • flex-direction: Establece la dirección en la que se alinearán los elementos flexibles dentro del contenedor.

  • justify-content: Define cómo se distribuyen los elementos flexibles a lo largo del eje principal del contenedor.

  • align-items: Define cómo se alinean los elementos flexibles a lo largo del eje transversal del contenedor.

  • align-content: Similar a justify-content, pero se aplica a múltiples líneas de elementos flexibles.

  • flex-wrap: Define si los elementos flexibles deben envolverse cuando no caben en una sola línea.

Estas propiedades te permiten controlar la disposición y alineación de los elementos flexibles dentro de su contenedor de manera efectiva.

Flex grow

El número X indica cuantas unidades de crecimiento crecerá el item.
Por ejemplo, flex-grow: 3; significa que el item crecerá 3 unidades.
Para saber cuanto crecerá un elemento debemos calcular a cuanto corresponde una unidad de crecimiento.

Flex shrink

Al contrario que la propiedad anterior, con flex-shrink se especifica el comportamiento de los items cuando el tamaño del contenedor es menor al de los ítems que lo componen. Por ejemplo, flex-shrink: 3; significa que el item reducirá 3 unidades.

Para saber cuanto se reducirá un elemento debemos calcular a cuanto corresponde una unidad de reducción.

La formula para calcular el tamaño de una unidad de reducción es la siguiente: unidad de reducción = espacio faltante / suma de flex-shrink
