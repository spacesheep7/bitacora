Frameworks


Los frameworks se utilizan en el ámbito de la programación de aplicaciones desde hace décadas. Recientemente se han usado para programar y diseñar aplicaciones web, por lo que ya existen decenas de frameworks para CSS. Que son

• Un framework es un conjunto de herramientas, librerías, convenciones y buenas prácticas que pretenden encapsular las tareas repetitivas en módulos genéricos fácilmente reutilizables.

Materialize


Ventajas de Materialize CSS:

  • Facilidad de uso: Permite diseñar sitios web atractivos con menos esfuerzo y código.
  • Responsividad: Ofrece un diseño responsivo por defecto, adaptándose a diferentes dispositivos.
  • Componentes predefinidos: Proporciona una amplia variedad de componentes listos para usar, como tarjetas, modales, formularios, etc.
  • Grid System: Utiliza un sistema de cuadrícula flexible que facilita el diseño y la disposición de elementos en la página.
  • Personalización: Permite personalizar fácilmente el aspecto de los elementos mediante clases predefinidas o reglas CSS personalizadas.
