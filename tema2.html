Tema 2: Propiedades personalizadas


Una propiedad personalizada en CSS, también conocida como variable CSS o variable personalizada, es un tipo de variable definida por el usuario que puede contener un valor específico y ser reutilizada en todo el documento CSS. Estas variables son definidas por el usuario y pueden tener cualquier nombre que desees, comenzando con dos guiones seguidos por el nombre de la variable


Nombre de variables


Se declaran con la siguiente sintaxis: -nombre: valor; • Y se invocan mediante la función var(), por ejemplo propiedad: var (-nombre);
• Los nombres de las variables CSS distinguen entre mayúsculas y minúsculas, por ejemplo:
--my-color
--My-color
• Son variables distintas. • Es importante el uso de identificadores semánticos. Por ejemplo:
~main-text-color
• De esta forma será más fácil de entender que:
-color

Alcance de variables



El alcance (ámbito) en el que podremos usar la variable, dependerá del selector donde se declare la variable.• Una buena práctica es declarar variables en la pseudo-clase root, y asi aplicarlas globalmente al documento HTML: root f -main-bg-color: brown;
