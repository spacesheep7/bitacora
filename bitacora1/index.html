Tema 1: Js conceptos basicos

JavaScript es un lenguaje de programación orientado a objetos derivado del estándar ECMAScript. Fue inicialmente desarrollado por Brendan Eich de Netscape bajo el nombre de Mocha, luego renombrado a LiveScript y finalmente JavaScript. Su cambio de nombre coincidió con la adición de compatibilidad con Java en el navegador de Netscape en 1995. Aunque su nombre sugiere una relación con Java, tienen propósitos y semánticas diferentes. En 1997, fue propuesto como estándar por la ECMA y adoptado como ECMAScript, también como estándar ISO poco después. JavaScript tiene una sintaxis similar a C pero adopta convenciones de Java. Se utiliza principalmente en el lado del cliente de las aplicaciones web para mejorar la interfaz de usuario y la interactividad. Desde 2012, todos los navegadores modernos admiten ECMAScript 5.1, mientras que los más antiguos al menos soportan ECMAScript 3. Se puede integrar JavaScript en HTML directamente o en archivos separados para su funcionamiento conjunto.

Agregar JavaScript directamente

Para insertar JS de forma directa en HTML se utiliza la etiqueta script script, el código JS debe quedar dentro de esta etiqueta. El código IS se puede colocar dentro de: La etiqueta head La etiqueta «body» Dependiendo de dónde se agregue el código JS en el archivo HTML, la carga será diferente. Por lo general se recomienda agregario en el «head» para que permanezca separado del contenido HTML Pero colocarlo dentro de «body puede ayudar a mejorar la velocidad de carga, ya que el contenido del sitio web se cargará más rápido, y solo después de eso se procesará el JavaScript.

Agregar JavaScript en un archivo separado

Insertar JavaScript directamente en HTML puede no ser la mejor opción, especialmente si se necesitan scripts en múltiples páginas. Es preferible mantener el código JS en archivos separados y referenciarlos de manera similar a los documentos CSS. Algunos beneficios de esta práctica incluyen la separación de HTML y JavaScript para una mayor sostenibilidad y reutilización, una mejor legibilidad y mantenimiento del código, y un rendimiento mejorado del sitio web gracias a la caché de los archivos JavaScript

Variables en Java

Las variables en JavaScript son contenedores para almacenar valores de datos y deben tener nombres únicos llamados identificadores, que pueden ser cortos o descriptivos. Se utilizan var, let y const para crear variables.

var: Se utiliza para declarar una nueva variable y asignarle un valor.

let: Indica que la variable puede ser reasignada a un valor diferente.

También puede declarar una variable sin asignarle un valor, en cuyo caso se inicializa como undefined.

const: Abreviatura de "constante", se utiliza para declarar una variable cuyo valor no puede ser reasignado. Intentar reasignar una variable const resultará en un TypeError.
