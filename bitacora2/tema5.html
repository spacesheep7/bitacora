Tema 5: zona de contacto


La "zona de contacto" dentro de una página web se refiere a la sección o área designada para que los visitantes interactúen y se comuniquen con el propietario del sitio web.

Esta área generalmente incluye elementos como formularios de contacto, información de contacto como dirección de correo electrónico, número de teléfono, dirección física, enlaces a redes sociales, entre otros.


Formularios adaptativos


Al crear formularios adaptativos en CSS, queremos asegurarnos de que los elementos del formulario se ajusten y se vean bien en diferentes tamaños de pantalla.

  • Unidades relativas: Utiliza porcentajes o vw para tamaños de elementos.
  • Flexbox/Grid: Usa Flexbox o CSS Grid para alinear y distribuir elementos de formulario.
  • Media Queries: Ajusta el diseño según el tamaño de la pantalla con Media Queries.
  • Botones táctiles: Aumenta el tamaño de los botones para dispositivos táctiles.
  • Legibilidad y accesibilidad: Asegúrate de que los elementos del formulario sean legibles y accesibles.

Formulario con boton


Este nos brinda la oportunidad de traer una presentacion diferente ante el usuario de una manera mas ordenada y atractiva, por medio del uso de escripts.







Label

La etiqueta (label) en HTML se utiliza para asociar un texto descriptivo a un elemento de formulario, como un campo de entrada (input).

Esta asociación mejora la accesibilidad y la usabilidad del formulario, ya que permite a los usuarios hacer clic en la etiqueta para activar el campo de entrada correspondiente
